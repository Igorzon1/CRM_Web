## Visão Geral
Este é um sistema web de CRM (Customer Relationship Management) construído com foco em arquitetura Enterprise e padrões de mercado, idealizado para compor um portfólio de alta qualidade.

## Tecnologias
- **Backend:** Java 17+, Spring Boot 3, Maven, PostgreSQL.
- **Frontend:** React (Vite), TailwindCSS.

Para mais detalhes sobre as regras de negócio, consulte a pasta `docs/product/`.
